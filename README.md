#  Echo-sense

EchoSense is an acoustic distance measurement system that computes the distance of the sound source based on microphone input and signal processing.

---

## 🚀 Features

* Audio sampling
* Distance estimation (intensity and frequency of sound)
* Calibration system for accuracy
* Graphs and statistics display
* Data export capabilities

---

## 🛠️ Technology Stack

Python, NumPy, SciPy, Matplotlib, PyQt5, SoundDevice

---

## 🔓 Installation

```bash
git clone https://github.com/Udit2027/Echo-sense.git
cd Echo-sense
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install numpy scipy matplotlib sounddevice pyqt5
```
